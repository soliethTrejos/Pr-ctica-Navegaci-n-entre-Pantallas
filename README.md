# CandyFocus

## Práctica: Navegación entre Pantallas

| Datos de la práctica | Información |
|---|---|
| Asignatura | POO II – Desarrollo de aplicaciones móviles en Android |
| Tema | Navegación entre pantallas |
| Tecnología | Kotlin + Jetpack Compose |
| Entregable | Proyecto funcional en Android Studio + repositorio público en GitHub |

---

## Objetivo

| Punto | Descripción |
|---|---|
| Objetivo principal | Implementar un sistema de navegación entre pantallas en una aplicación móvil utilizando Jetpack Compose. |
| Propósito | Comprender el flujo de navegación y la estructura de múltiples pantallas dentro de una app Android. |

---

## Contenido practicado

| Contenido | Aplicación en el proyecto |
|---|---|
| Navegación entre pantallas | Se implementó navegación usando `NavHost`, `NavController` y rutas para cada pantalla. |
| Flujo de usuario | El usuario puede moverse entre Inicio, Detalle y Perfil. |
| Manejo de múltiples vistas | La app tiene tres pantallas separadas y funcionales. |
| Componentes interactivos básicos | Se usan botones, campos de texto, temporizador, selección de imagen y tarjetas interactivas. |

---

## Pantallas de la aplicación

| Pantalla | Descripción | Interacción |
|---|---|---|
| Pantalla 1: Inicio | Muestra un temporizador de concentración tipo Pomodoro. | Permite iniciar, pausar y reiniciar el temporizador. |
| Pantalla 2: Detalle | Muestra una lista de tareas de enfoque. | Permite agregar tareas y marcarlas como completadas. |
| Pantalla 3: Perfil | Muestra el resumen del usuario y su progreso. | Permite cambiar la foto de perfil, ver actividades y tiempo de enfoque. |

## Funcionalidades principales

| Funcionalidad | Descripción |
|---|---|
| Temporizador | Permite trabajar con sesiones de concentración de 25 minutos. |
| Lista de tareas | Permite agregar tareas y marcarlas como completadas. |
| Perfil de usuario | Muestra nombre, foto y resumen de productividad. |
| Foto de perfil | Permite seleccionar una imagen desde la galería y conservarla al volver a abrir la app. |
| Gráfico de actividades | Se despliega al presionar el botón del trofeo en el perfil. |
| Tiempo de concentración | Se despliega al presionar el botón del rayo en el perfil. |
| Barra de navegación inferior | Permite cambiar entre Inicio, Detalle y Perfil. |
